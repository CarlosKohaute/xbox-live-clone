# API Xbox live clone

API desenvolvida utilizando Nest.js.

## Como fazer o clone

```bash
git clone git@https://github.com/CarlosKohaute/xbox-live-clone
```

## Como instalar as dependências

```bash
npm install
```

## Como rodar o projeto

### Produção

```bash
npm run start
```

### Desenvolvimento

```bash
npm run start:dev
```
